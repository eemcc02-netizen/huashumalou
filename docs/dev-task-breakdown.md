# 功能任务拆解

本文档把用户规划的 6 个功能拆成可执行任务，方便逐步开发。

## 1. 回车发图片

当前状态：已完成第一版，待验证和增强。

已完成：

- 新增 `imageAutoSendStrategy`。
- Popup 和设置页可切换 `自动发送 / 回车发送`。
- Content Script 根据策略执行点击确认或回车确认。
- 版本号更新到 `1.9.2`。

待办：

- 在真实目标页面验证回车发送。
- 若 Enter 无效，补站点级发送按钮点击。
- 优化 Popup 和设置页小三角样式。

## 2. 图片话术

优先级：高。

任务：

- 定义图片话术数据结构。
- 设置页新增图片话术创建入口。
- 支持上传/粘贴图片。
- 支持图片预览。
- 支持保存、编辑、删除。
- 候选面板展示图片话术标识。
- 快捷词触发后将图片写入剪贴板并粘贴到当前输入框。
- 单条图片话术支持 `插入后自动发送`。
- 导入导出兼容图片话术。

风险：

- 浏览器剪贴板图片权限和不同站点粘贴行为不一致。
- 图片 base64 可能导致 storage 变大。

## 3. 监听

优先级：中。

任务：

- 梳理现有输入监听、keydown、input、focusin、MutationObserver。
- 抽象监听配置。
- 支持站点级规则。
- 支持发送按钮 selector。
- 支持确认按钮文案。
- 增加调试日志开关。

风险：

- 监听过多可能影响页面性能。
- 不同站点 DOM 结构差异大。

## 4. 更多自定义

优先级：中。

任务：

- 梳理现有可配置项。
- 增加站点级覆盖。
- 增加候选面板显示配置。
- 增加图片发送策略站点覆盖。
- 增加快捷词匹配方式配置。
- 补导入导出兼容。

## 5. 文本热分隔

优先级：中高。

任务：

- 新增设置字段。
- 设置页新增配置 UI。
- 文本话术插入前处理分隔符。
- 支持转换为换行。
- 支持 trim 和保留空段配置。
- 补手动回归用例。

建议先做原因：

- 改动相对集中。
- 不依赖图片剪贴板权限。
- 对客服长话术体验提升明显。

## 6. 悬浮球

优先级：中后。

任务：

- 新增全局开关。
- Content Script 渲染悬浮球。
- 支持拖拽和位置保存。
- 点击展开迷你面板。
- 迷你面板支持搜索话术。
- 支持插入文本话术。
- 支持插入图片话术。
- 支持站点级禁用。

风险：

- 容易遮挡业务页面。
- 样式隔离需要使用 Shadow DOM。
- 需要和现有候选面板、备忘录悬浮窗协调层级。

## 建议开发顺序

1. 稳定 `回车发图片`。
2. 实现 `图片话术`。
3. 实现 `文本热分隔`。
4. 做 `监听` 和 `更多自定义`。
5. 最后做 `悬浮球`。
