# Development Log

## 2026-07-09

### 图片话术第一版

- 在设置页话术表单中加入“话术类型”，支持文本话术与图片话术切换。
- 图片话术保留标题、快捷词、分类，并新增单张图片上传、预览、文件名、格式、大小展示。
- 图片转为 dataURL 后沿用 `tb-snippets` 数组存储，不新增独立存储表。
- 单张图片大小限制为 2MB，超过后提示用户压缩后再上传。
- 列表中图片话术显示类型标签、小缩略图和 `[图片] 文件名` 预览。
- 编辑图片话术时回填图片预览与“插入后自动发送”开关。
- 内容脚本支持命中图片话术后删除快捷词、写入剪贴板并向当前输入框派发粘贴。
- `autoSendAfterInsert` 与全局 `autoSendImageConfirm` 同时开启时，复用现有图片自动确认策略。
- JSON 导入导出保留图片话术字段，旧文本话术默认按 `type: "text"` 兼容。
