# 版本日志（Changelog）

本文件用于记录 `话术马喽（马喽版）` 的功能变更，便于后续版本管理、问题回溯和发布说明整理。  
建议每次发布前更新一次，并与 `manifest.json` 中的 `version` 保持一致。

---

## 记录规范

建议按以下分类记录：

- `Added`：新增功能
- `Changed`：行为调整/体验优化
- `Fixed`：问题修复
- `Removed`：移除功能

建议每条记录尽量写清楚：

- 用户可感知的变化
- 配置项或默认值变化
- 兼容性影响（如有）

---

## [Unreleased]

## [2.0.7] - 2026-07-23

### Changed

- “受害马喽”使用固定标签页池处理任务，不再为每条记录反复创建和销毁页面。
- CRM 采集模块提前到 `document_start` 加载，资料区域出现后立即提取，无需等待整页加载完成。
- 页面消息重试间隔由 500ms 缩短为 120ms。

## [2.0.6] - 2026-07-23

### Changed

- “受害马喽”的并发页面设置改为数字输入，允许设置 `1–10` 个并发任务。

## [2.0.5] - 2026-07-23

### Added

- “受害马喽”新增“复制全部”，复制内容包含昵称、ID 和 Base64 真实头像。

### Changed

- 腾讯文档直接写入与手动复制共用独立剪贴板模块，确保两条路径的数据格式一致。

## [2.0.4] - 2026-07-23

### Added

- 新增模块化“受害马喽”工具页，支持批量输入 CRM 用户 ID 并按队列采集。
- 新增独立 CRM 页面适配器，提取昵称、ID 和真实头像数据。
- 新增腾讯文档表格适配器，支持从指定起始单元格写入文字与内嵌头像。
- popup 新增与“启动顺风耳”并排的“受害马喽”入口。

### Changed

- 版本号更新为 `2.0.4`。
- 采集任务、页面提取和腾讯文档写入拆分为独立模块，避免继续扩大现有主脚本。

## [1.9.3] - 2026-07-09

### Added

- 新增图片话术第一版：支持图片存储、设置页新增/编辑/删除、单张图片插入。

### Fixed

- 修复设置页打开异常与部分页面文字乱码问题。
- 优化 SCRM 页面图片插入兼容性：在剪贴板粘贴之外增加隐藏图片上传框 fallback。

### Added

- 新增“图片话术”第一版：支持在设置页上传单张图片、预览图片信息，并存入现有 `tb-snippets` 数组。
- 图片话术支持 `type: "image"`、`imageName`、`imageMime`、`imageSize`、`imageData`、`autoSendAfterInsert` 字段。
- 内容脚本支持快捷词命中后删除当前快捷词、写入图片剪贴板并向当前输入框粘贴单张图片。
- 图片话术可按单条配置“插入后自动发送”，并复用全局图片自动确认的点击/回车策略。

### Changed

- `normalizeSnippets` 兼容旧文本话术，缺失 `type` 时默认按 `type: "text"` 处理。
- JSON 导入导出保留图片话术字段，旧 JSON 文本话术导入保持兼容。

## [1.9.2] - 2026-07-09

### Added

- 图片自动确认新增策略切换：支持“自动发送”与“回车发送”模式，并在 popup 与设置页可直接切换。

### Changed

- 图片自动发送策略默认保持“自动发送”，并兼容旧设置自动回退。

### Added

- 新增“快捷点击（调试选项）”：支持在设置页配置快捷键，按元素 selector 或页面坐标自动点击目标。
- 新增页面选取模式：可从目标网页点选 button/div 或坐标，并自动回填到设置页。
- 新增 AI 服务商预设：`MiniMax 国内`、`MiniMax 国际`、`DeepSeek 官方`、`火山引擎 Ark`。
- 新增 AI 模型候选项：`deepseek-v4-flash`、`deepseek-v4-pro` 以及火山引擎 endpoint 占位示例。
- 新增可编辑的 `User Prompt 模板`（支持 `{{context}}` 与 `{{count}}` 变量）。

### Changed

- AI 基础配置页中，服务商选择与接口地址联动逻辑增强：
  - MiniMax 国内/国际支持按当前格式（OpenAI/Anthropic）切换默认地址。
  - DeepSeek 支持 OpenAI/Anthropic 双格式地址联动。
  - 火山引擎固定 OpenAI 兼容地址。
- `/ai` 相关上下文角色文案统一为“我 / 用户”，避免“老师 / 学员”语义冲突。

### Fixed

- 修复 MiniMax 接口 404 场景下的候选端点回退问题，提升连通性与容错。
- 修复 AI 建议结果中的噪声内容清洗问题：
  - 过滤 `<think>...</think>` 思考块
  - 过滤“上下文回显”类文本（如“上下文：”“我：...”等）
  - 保留可直接发送的建议句
- 修复 popup 验证面板角色计数展示字段不一致问题（`me/user`）。

## [1.8.0] - 2026-04-27

### Added

- 新增联想模板显示模式（仅标题 / 仅内容 / 标题+内容）。
- 新增联想预览字数控制（0-30）。
- 新增填入特效颜色设置（主色 / 辅色）。
- 新增两种填入特效：`霓虹电弧版`、`流光羽焰版`。

### Changed

- 联想面板交互优化：键盘上下选中时自动滚动到可视区域。
- AI 建议与快捷补全在面板中展示逻辑进一步区分，提升客服高频场景可读性。

### Fixed

- 修复长候选列表中仅高亮变化、滚动条不跟随导致的“选中项定位困难”问题。
